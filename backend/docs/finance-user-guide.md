# Finance user guide

Create one invoice for the obligation, then apply partial or full payments to that invoice. A payment must never be used to create revenue or VAT. Verify receipt number, account, amount, method, and date before saving. Corrections use approved reversal/refund workflows; never delete posted entries. Daily close requires cash/bank totals, open receivables, VAT, trial balance, and reconciliation review. Escalate every mismatch before continuing postings.
